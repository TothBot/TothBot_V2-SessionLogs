CHANGES to logger.py:

  CHANGE 1 - Raise timeout. Line 166:

    BEFORE:  ALERT_SMTP_TIMEOUT_SEC: int = 5
    AFTER:   ALERT_SMTP_TIMEOUT_SEC: int = 20

    Rationale: 5 sec covers four network round-trips
    (connect, starttls, login, sendmail) on a single
    budget, any one of which can block >5 sec on
    transient server load. 20 sec absorbs realistic
    transient without blocking excessively.

  CHANGE 2 - Wrap SMTP conversation in a 2-attempt
             retry with 1-sec backoff between attempts.
             Lines 296-316 (approximate; reverify in
             fresh read).

      ALERT_SMTP_RETRY_BACKOFF_SEC: int = 1

      sent = False
      last_exc = None
      for attempt in range(1, ALERT_SMTP_MAX_ATTEMPTS + 1):
          try:
              with smtplib.SMTP(
                  smtp_host, smtp_port,
                  timeout=ALERT_SMTP_TIMEOUT_SEC,
              ) as server:
                  server.starttls()
                  server.login(smtp_user, smtp_pass)
                  server.sendmail(
                      smtp_user,
                      [ALERT_EMAIL],
                      msg.as_string(),
                  )
              sent = True
              break
          except Exception as exc:
              last_exc = exc
              if attempt < ALERT_SMTP_MAX_ATTEMPTS:
                  time.sleep(
                      ALERT_SMTP_RETRY_BACKOFF_SEC
                  )

      if sent:
          # existing ALERT_SENT emission path
          ...
      else:
          # existing stderr + ALERT_SMTP_FAILED path
          # raise last_exc-bound logging
          ...
