   5b. tothbot/exchange/ws_manager.py [MOD] - wired the sole writer. WSManager holds the
       PositionMirror and is the ONLY caller that tags a write with WRITER_ID:
       record_execution() + restore_position_mirror() are the write surface; position()/
       has_position()/open_position_symbols()/open_positions() are the read helpers other
       modules use (HR-PM-009 read contract). The deferred shell comment was updated.
   5c. tothbot/exchange/assembler.py [NEW] - the PATH-2 startup assembler (async edge).
       DataLayerAssembler.build() opens N public sockets from a ShardPlan and constructs,
       per shard, a ConnectionKeepalive, the per-pair SilentPairMachine registry, a
       DispatchTable, and a ShardReceiveLoop bound to the SHARED ShardReconnectCoordinator
       (one HR-WM-012 gate) + the SHARED ReconnectDriver; then runs the initial PACED
       subscribe (subscribe_requests -> to_wire, one RPC per global channel + per
       pair x per-pair channel = assignment.subscribe_count) over the process-singleton
       SubscribeTokenBucket (WM-PACE-001, NO bypass). On a shard's local drop the driver
       reopens that shard's socket and run_step re-arms its silent pairs
       (mark_shard_reconnect) + re-subscribes on the fresh socket (RESUME_KEEPALIVE resets
       the keepalive). Public shards carry no private channels, so the restore set is
       public-only regardless of mode (the private connection is a separate single
       connection, assembled later). DataLayer.run()/stop() fan the shard loops out.
   5d. tothbot/exchange/reconnect_driver.py [MOD] - the single shared driver is now

       coordinator) drives N shards - open_socket(shard) opens that shard's fresh socket,
