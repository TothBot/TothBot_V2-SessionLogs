filesystem tree rooted at its launch directory. It does

mechanism Claude Chat uses). Co-locating session logs with
the two git repos under the TothBot parent folder gives
Claude Code access to all required files (governance docs,

tree when launched from C:\Users\wstot\GitHub\TothBot.

FP/DP ANALYSIS (performed this session):
  Option A — revert the folder move:
    Requires Claude Code launch from C:\Users\wstot\.
    Exposes entire user profile (.ssh keys, Documents,
    Downloads, AppData, Desktop, browser profiles).
    Violates least-privilege. Larger scan tree. Multiple
    possible launch dirs → error-prone.

  Option B — ratify the new location (update TB00000):
    Limits Claude Code scope to TothBot content only.
    Three subfolders: two git repos + session logs.
    Least-privilege preserved. Faster. Single canonical
    launch directory. One-time DC cost: TB00000 v2_8 →
    v2_9, TB00103 v1_0 → v1_1, TB00103_NSI v1_0 → v1_1.

DECISION: Option B. Efficiency, reliability, and safety
all improve. Governance updated in this session to match
operational reality.
