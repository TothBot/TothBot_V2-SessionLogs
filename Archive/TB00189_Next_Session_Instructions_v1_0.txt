ARCHITECTURAL INTENT.
TothBot V2 operates as a parallel-module framework with
independent trading modules (Long_Module + Short_Module
+ any future trading module) as siblings above the WS
Manager data-acquisition layer. Per TB00000 v2_29 §9.6:
each module gets its own CIATS instance with its own
statistical pool, its own 200-trade inference threshold,
its own 600-trade per-regime threshold, and its own
parameter proposals. Cross-module data contamination is
prohibited. The 1011010 coding spec must be amended
from the legacy single-CIATS / Long-only design to the
per-module framework instantiation. This is the
architectural successor to the legacy design retired
at TB00000 v2_29.
