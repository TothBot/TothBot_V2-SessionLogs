  CHANGE 2 — AR-045 (TB00015):
    OHLC candle-close detection via interval_begin change.
    last_interval_begin tracked per symbol.
    Fire pipeline only on interval_begin change.
    Do NOT use wall clock for candle detection.

  CHANGE 3 — AR-046 (TB00015):
    emergSL triggers.reference="last" explicit in all
    batch_add calls. Never rely on defaults for crash
    protection parameters. Never use "index".

  CHANGE 4 — AR-047 (TB00015, amended TB00016):
    ALL price/qty rounding via Decimal.quantize() —
    NEVER math.floor/ceil. Float binary representation
    causes precision errors. All Kraken API values:
    Decimal(str(value)) immediately on receipt.
    Four rounding ops: entry ROUND_DOWN, qty ROUND_DOWN,
    TP ROUND_UP, emergSL trigger ROUND_DOWN.

  CHANGE 5 — AR-048 (TB00015):
    Exit Controller MAE uses bid price (not last) for
    long positions. bid = realizable exit value.
    Drawdown formula also uses bid. Consistent with
    event_trigger:bbo choice.

  CHANGE 6 — AR-049 (TB00015):
    Formal 10-step startup sequence. No pipeline until
    at least one READY pair. Full spec in 1011014.

  CHANGE 7 — AR-050 (TB00016):
    Balances channel: use wallets[type=spot][id=main]
    .balance ONLY. Top-level balance includes earn/
    staking wallet — NOT tradeable capital.

  CHANGE 8 — AR-051 (TB00016):
    Portfolio for SIZING = spot_usd_balance (cash only).
    Does NOT include MTM of open positions.
    tradeable_USD = portfolio_USD × tradeable_pct.
    per_trade_USD = tradeable_USD × per_trade_pct.

  CHANGE 9 — AR-052 (TB00016):
    Drawdown baseline = spot_usd_balance at startup.
    Set ONCE. NEVER reset on reconnect.
    Current portfolio (drawdown) = cash + bid×qty.
    Session pause 5%, full halt 10%. No auto-close.

  CHANGE 10 — AR-053 (TB00016):
    Pending Order Registry. Self-tracks USD committed
    to outstanding entry orders. Gate 7 checks
    available_USD = balance − pending commitments.
    Prevents overcommitment on simultaneous pipeline
    activations.

  CHANGE 11 — AR-047 amendment (TB00016):
    Original AR-047 used math.floor/ceil. Amended to
    Decimal.quantize() per BP-DEC-001 through -008
    in 1011001 Engineering_Best_Practices dv1_4.

  CHANGE 12 — AR-054 (TB00017):
    Entry order partial fill + expiry/cancel creates
    unprotected open position. exec_type=expired or
    canceled with cum_qty > 0: recalculate TP/emergSL
    from avg_price, dispatch batch_add, enter Position
    Mirror. If below qty_min/cost_min: market sell.

  CHANGE 13 — AR-055 (TB00017):
    Dead Man's Switch (cancel_all_orders_after)
    EXPLICITLY PROHIBITED. Cancels emergSL orders —
    destroys crash protection. Hard rule. Code review
    must reject any PR containing this call.

  CHANGE 14 — AR-056 (TB00017):
    Mid-session WS reconnect sequence (11 steps)
    distinct from startup. Never reset
    portfolio_baseline_USD or drawdown on reconnect.
