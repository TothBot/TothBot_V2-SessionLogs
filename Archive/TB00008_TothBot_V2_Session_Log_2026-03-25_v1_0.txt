3.6 Download Problem — .docx Corrupted in Browser
------------------------------------------------------------

PROBLEM:
  Bill reported the .docx file opened as gibberish regardless
  of browser (Chrome or Edge). Right-click → Save link as
  did not resolve the issue. The file appeared corrupted
  in every attempt.

ANALYSIS:
  The .docx binary file was not being received correctly
  by Bill's browser. This is a known issue with binary
  MIME types in some browser+platform combinations. The
  file itself was valid — the problem was the delivery
  mechanism.

RESOLUTION:
  Converted the .docx to PDF using LibreOffice headless:
    python3 soffice.py --headless --convert-to pdf
    Input:  0500000_System_Architecture_Overview_dv1_6.docx
    Output: 0500000_System_Architecture_Overview_dv1_6.pdf
    Size:   1.5 MB

  PDF downloads cleanly in all browsers. Bill can open
  in any PDF viewer (Chrome, Edge, Acrobat Reader).
  Bill confirmed the PDF was accessible.
