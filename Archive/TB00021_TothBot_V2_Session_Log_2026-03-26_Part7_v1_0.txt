------------------------------------------------------------
FINDING 1: RSI SMOOTHING FORMULA AMBIGUITY IN AR-068
------------------------------------------------------------

LOCATION: AR-068, 1011002 WM-SSS-001 through -008,
          0711001 SSS-001 through -012.

THE PROBLEM:
AR-068 stated: "RSI(14): first 14 closes →
RS=avg_gain/avg_loss. EMA-smoothed thereafter."

The phrase "EMA-smoothed" is ambiguous and points to the
wrong algorithm. Two different formulas both call
themselves EMA-like:

  Wilder's SMMA (RMA):  α = 1/14 = 0.0714
    avg = (prev × 13 + new) / 14
    Used by: TradingView rma(), StockCharts, Wikipedia,
             Supra, most professional platforms
    This is the CORRECT formula for RSI.

  Standard EMA:          α = 2/15 = 0.1333
    avg = prev × (1 - α) + new × α
    Used by: pandas ewm default, some commercial packages
    This is WRONG for RSI.

These produce materially different RSI values — confirmed
~5-8% variance at identical price data (AlphaRithms,
confirmed across 4 searches, 10+ sources). A coder
reading "EMA-smoothed" would likely implement standard
EMA (α=2/15), producing non-standard RSI that:
  - Does not match any trading platform's RSI display
  - Would systematically mis-fire SSS SC-SSS-1
    (RSI 50-70 gate) relative to market conventions

RESOLUTION — CORRECT RSI(14) INCREMENTAL FORMULA:

  Seed (first 14 candles):
    avg_gain = sum(gains[:14]) / Decimal("14")
    avg_loss = sum(losses[:14]) / Decimal("14")

  Incremental (each subsequent candle close):
    avg_gain = (prev_avg_gain × 13 + current_gain) / 14
    avg_loss = (prev_avg_loss × 13 + current_loss) / 14

  RS  = avg_gain / avg_loss
  RSI = 100 − (100 / (1 + RS))

IMPORTANT DISTINCTION — EMA(9) and EMA(21) use STANDARD EMA:
  EMA(9):  α = 2/10 = 0.2000  (standard EMA — CORRECT)
  EMA(21): α = 2/22 = 0.0909  (standard EMA — CORRECT)
  Only RSI uses Wilder's SMMA. Do not conflate.

EVIDENCE BASE: 10+ independent sources across 4 sequential
searches — TradingView (rma(gain,14)), StockCharts.com
(explicit Wilder SMMA formula with (prev×13+current)/14),
Supra Oracle docs, Wikipedia RSI page, LuxAlgo, AlphaRithms,
Quantinsti, EliteTrader forum, PineScriptMarket, Medium.
All confirm Wilder's SMMA as correct RSI algorithm.
A 14-period SMMA is equivalent to a 27-period standard
EMA (2×14-1=27), not a 15-period EMA.

LOSS MINIMIZATION: Incorrect RSI formula → incorrect
SC-SSS-1 evaluation → entries in wrong momentum conditions
→ directly violates primary mission.
